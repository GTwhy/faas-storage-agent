# 使用说明
## 依赖项安装
#### 安装Django
```
pip3 install django
```
#### 安装Django OAuth Toolkit
```
pip3 install django-oauth-toolkit
```
## 启动认证服务器
```
python3 manage.py runserver localhost:10087
```
